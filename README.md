# Bereket Fikre Portfolio Website

Professional creative designer portfolio website showcasing brand identity, UI/UX design, graphic design, and digital experiences.

## 🚀 Features

- **Fully Responsive** - Optimized for all screen sizes (mobile, tablet, desktop)
- **SEO Optimized** - Complete meta tags, structured data, and sitemap
- **Performance Optimized** - Fast loading, lazy loading images, optimized assets
- **Modern Design** - Clean, professional monochrome design
- **Interactive Elements** - Smooth animations and transitions
- **Accessibility** - Alt text for all images, semantic HTML

## 🛠️ Tech Stack

- React 18.2.0
- React Scripts 5.0.1
- CSS3 with modern features
- WebP images for optimal performance

## 📦 Installation

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## 🚀 Deployment

### GitHub Pages Deployment

1. **Install gh-pages** (if not already installed):
   ```bash
   npm install --save-dev gh-pages
   ```

2. **Build and deploy**:
   ```bash
   npm run build
   npm run deploy
   ```

3. **Configure GitHub Pages**:
   - Go to your repository settings
   - Navigate to Pages section
   - Select source: `gh-pages` branch
   - Your site will be live at: `https://bereketfikre2021-sudo.github.io/Portfolio`

### Manual Git Deployment

```bash
# Initialize git (if not already done)
git init

# Add remote repository
git remote add origin https://github.com/bereketfikre2021-sudo/Portfolio.git

# Stage all files
git add .

# Commit
git commit -m "Deploy portfolio website"

# Push to GitHub
git push -u origin main
```
